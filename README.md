This project is a student records management database developed for Midlands Community College to improve the way student information is stored, organized, and accessed. 
The system centralizes important student data, including registration details, medical information, and academic records, into a structured and reliable database.
The goal of this project was to replace fragmented or manual record-keeping processes with a centralized digital system that improves data accuracy, accessibility, 
and administrative efficiency.

Features
Centralized storage of student information
Management of student registration records
Storage of student medical information
Tracking of academic records and performance
Structured tables with defined relationships to maintain data integrity
Organized data retrieval for administrative use

Database Design
The database was designed using relational database principles to ensure efficient data storage and consistency. 
Tables were structured to reduce redundancy and maintain clear relationships between different types of student information.

Key components include:
Student personal and registration information
Medical records linked to individual students
Academic records and course performance
Structured relationships between entities to maintain data consistency

Purpose:
The system was developed to:
Improve record management and accessibility
Reduce data duplication and manual errors
Provide a structured system for storing sensitive student information
Support administrative staff in managing student data more efficiently

Technologies Used:
Microsoft Access
Relational database modeling
Data normalization principles
